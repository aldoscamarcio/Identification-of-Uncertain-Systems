State Estimation for an Industrial AGV
Developed as a team project for the "Identification of Uncertain Systems" course.

The project focuses on estimating the state of an industrial Automated Guided Vehicle (AGV). The vehicle's pose in space is defined by its spatial coordinates x and y, and its orientation θ. Given the availability of multiple sensor readings and the nonlinear dynamics of the AGV, advanced estimation techniques were required. Specifically, we implemented and compared an Extended Kalman Filter (EKF) and an Unscented Kalman Filter (UKF). Finally, the estimated trajectory was smoothed using a Rauch-Tung-Striebel (RTS) smoother to improve overall accuracy.

The entire project was developed and simulated using MATLAB/Simulink.
